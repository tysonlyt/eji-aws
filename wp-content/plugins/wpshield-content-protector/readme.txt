== Installation ==
1. Install WP Shield Content Protector Lite either via the WordPress.org plugin repository or by uploading the files to your server.
2. Activate WP Shield Content Protector.
3. Navigate to the WP Shield tab at the middle of your admin menu and click the "Settings" button.
