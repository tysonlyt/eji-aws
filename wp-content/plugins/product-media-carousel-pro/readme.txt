== Description ==

Product Media Carousel is a powerful WordPress plugin that enhances your WooCommerce product pages with beautiful, responsive carousels that combine product images and YouTube videos seamlessly.

= Free Version Features =

* **YouTube Video Integration** - Add unlimited YouTube videos to your product galleries
* **Elementor Integration** - Full integration with Elementor page builder
* **Multiple Carousel Effects** - Choose from Slide, Fade, Cube, Coverflow, and Flip effects
* **Customizable Navigation** - 5 different navigation styles with full color control
* **Thumbnail Positioning** - Place thumbnails on top, bottom, left, or right
* **Lightbox Support** - Built-in Fancybox integration for image and video viewing
* **Responsive Design** - Fully responsive and mobile-friendly
* **Theme Integration** - Automatically adapts to your Elementor theme colors
* **Arrow Sorting** - Easily reorder media items with up/down arrows
* **Product ID Column** - View product IDs in WooCommerce product list

= Pro Version Features =

* **All Free Features** - Everything from the free version
* **Vimeo Video Support** - Add Vimeo videos to your galleries
* **Self-Hosted Videos** - Upload and use your own video files (MP4, WebM)
* **Video File Upload** - Direct upload to WordPress media library
* **🌟 Bulk Video Import (CSV)** - Import videos for multiple products at once
* **🌟 Manual Bulk Input** - Add videos to multiple products with product search
* **🌟 Product Name Search** - Search products by name (no need to remember IDs)
* **🌟 Import History** - Track all your bulk import operations
* **Import Progress** - Real-time progress display during bulk import
* **Detailed Logs** - See exactly which products succeeded or failed

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/product-media-carousel` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Make sure WooCommerce and Elementor are installed and activated.
4. Edit any WooCommerce product and find the "Product Media Gallery" meta box.
5. Add YouTube videos and configure your carousel settings.
6. Use the Elementor widget to display the carousel on your product pages.

== Frequently Asked Questions ==

= Does this plugin require WooCommerce? =

Yes, this plugin is specifically designed for WooCommerce products.

= Does this plugin require Elementor? =

While the plugin works best with Elementor, you can also use the shortcode `[product_media_carousel]` to display carousels without Elementor.

= How many videos can I add per product? =

You can add unlimited YouTube videos to each product.

= Can I customize the carousel appearance? =

Yes! The plugin offers extensive customization options including navigation styles, colors, sizes, effects, and more.

= Does it work with product variations? =

Currently, the carousel displays the same media for all variations. Product variation support is coming in a future update.

= Is it mobile responsive? =

Yes, the carousel is fully responsive and works perfectly on all devices.
