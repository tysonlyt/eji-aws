Create a custom product page that allows selecting products from the product library and integrates with TI Wishlist.

== Description ==

Unique Client Page 是一个强大的WordPress插件，允许您创建自定义产品页面，从产品库中选择产品，并与TI Wishlist无缝集成。

主要功能：
- 响应式产品网格和列表视图
- 产品筛选和搜索功能
- 与TI Wishlist集成
- 产品详情弹窗
- 可定制的产品展示
- 支持WooCommerce产品

== 安装 ==

1. 上传 `unique-client-page` 文件夹到 `/wp-content/plugins/` 目录
2. 在WordPress后台的插件页面激活插件
3. 使用短代码 `[unique_client_products]` 在任何页面或文章中添加产品展示

== 常见问题 ==

= 这个插件需要WooCommerce吗？
是的，这个插件需要安装并激活WooCommerce才能正常工作。

= 如何自定义产品展示样式？
您可以通过编辑插件目录下的CSS文件来自定义样式，或者添加自定义CSS到主题的样式表中。
